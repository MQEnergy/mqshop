# mqshop 麻雀电商系统
麻雀电商系统 基于go-skeleton + Reactjs + shadcn-UI 开发的面向出海的插件化后台管理系统

技术栈：
1、后端骨架：
[go-skeleton](https://github.com/MQEnergy/mqshop)
2、前端组件：
[shadcn-ui](https://github.com/shadcn-ui/ui) 

## 运行项目
1、前端（待开发）
```shell
cd web
pnpm install
pnpm run dev
```

2、后端
将数据库导入到mysql中、配置configs 然后运行：
```shell
air
```
